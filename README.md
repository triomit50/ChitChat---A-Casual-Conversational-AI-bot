# 🤖 ChitChat - A chatbot for a casual convo

## 📑 About:
A chatbot, to conduct simple conversation and easy searches of information and data. Chatbot engineered with llama-3.1-8b-instant model from Groq AI tool, to retrieve the API token. 

With the high quality of word selection and sentence structuring from the Llama model, this chatbot has been equipped with chat history storage capacity, to store the previous messages as history data and hence, use them as reference for responding to the latest prompts from the user.

## 🛠️ Tools used:
- VSCode (as IDE)
- Python (app development)
- Jupyter Notebook (model testing and exploration)
- Streamlit (easy deployment as frontend in the localhost ; testing)
- Langchain (framework to handle the prompt-model chain)
- LangSmith (tracing the performance of my model)
